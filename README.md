# Bastard Crew Games

A starter personal website for showcasing TTRPG books and writing process notes.

## How to Open

Open `index.html` in a browser. No build step is required.

## Where to Edit

- `index.html` controls the main page structure and long-form copy.
- `styles.css` controls the visual design.
- `script.js` contains the sample books and journal entries.

Replace the sample creator name, book titles, journal posts, and newsletter placeholder with your own details when you are ready.
